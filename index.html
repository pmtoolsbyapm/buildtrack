🔒 Rules that apply to all imports:
✓ Save as CSV before uploading — In Excel: File → Save As → CSV (Comma delimited). In Google Sheets: File → Download → CSV.
✓ Keep your column headings in Row 1 — They must match exactly what is shown below. Do not rename them.
✓ No merged cells — Every cell must be its own individual cell. Merging columns or rows will break the import.
✓ No blank rows — Do not leave empty rows between your data. Data must be continuous from Row 2 downward.
✓ Dates must be YYYY-MM-DD — Write 2025-08-15, not 15/08/2025 or Aug 15. Any other format will be ignored.
✓ Numbers only in amount columns — No currency symbols or commas. Write 320000 not $320,000. Currency symbol is set in your project settings.
